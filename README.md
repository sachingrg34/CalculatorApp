

Calculator App

Project Description

This Android Calculator App, developed with Java and Android Studio, provides real‑time arithmetic calculations and a clean, responsive interface. 
It was completed as part of the CST Android development assignment, focusing on improving skills in layout design, user interaction handling, and core Java logic.


Features Implemented

Basic arithmetic operations:



1. Addition (+)

2. Subtraction (−)

3. Multiplication (×)

4. Division (÷)

5. Number buttons (0–9)

6. Operator buttons (+, −, ×, ÷)

7. Equals (=) button to calculate results

8. Clear (C) button to reset the calculator

9. Division‑by‑zero error handling

10. Real‑time display updates

11. Clean and responsive UI using GridLayout

12. Smooth button interactions

How to use the App

1. Open the app on your Android device or emulator.

2. Tap the number buttons to enter the first value.

3. Select an operator (+, −, ×, ÷).

4. Enter the second value.

5. Press = to view the result.

6. To continue, add an operator, enter the next value, and press = again.

7. Press C to clear the display and start a new calculation.

8. If you try to divide by zero, the app will display an error message.


Additional Features

1. Decimal point support

2. Backspace button

3. Percentage (%) button

4. Plus/Minus (±) toggle
